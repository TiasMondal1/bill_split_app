## Getting Started

1. Install Flutter dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

## Platform Setup

### Android
- Minimum SDK: 21
- Add camera and storage permissions in `android/app/src/main/AndroidManifest.xml`

### iOS
- Minimum iOS: 12.0
- Add camera and photo library usage descriptions in `ios/Runner/Info.plist`
